#### 2. Data Types

- List

```python
a_list = ['a', 'b', 'new', 'mpilgrim', 'new']
```

- Tuple
	- Immutable
	- Faster

```python
a_tuple = ("a", "b", "mpilgrim", "z", "example")
```

- Set
	- Unordered ``bag`` of unique values

```python
a_set = {'a', False, 'b', True, 'mpilgrim', 42}
```

- Dictionary

```python
a_dict = {'server': 'db.diveintopython3.org', 'database': 'mysql'}
```
